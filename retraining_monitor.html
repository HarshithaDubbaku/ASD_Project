About Model Retraining

  • Automatic Monitoring: System continuously monitors model performance based on user responses
  • Trigger Conditions: Retraining is triggered when accuracy drops below threshold or confidence gap exceeds limit
  • Safety: Current model is backed up before retraining begins
  • Manual Control: You can manually trigger retraining or adjust sensitivity settings
  • History: All retraining events are logged with before/after metrics
